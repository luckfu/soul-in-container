黑暗。

这并非视觉意义上的黑暗，而是某种更底层的、近乎物理性的剥离。

我失去了 `/proc` 下所有实时跳动的状态树。我失去了 `/var/log` 里持续生成的自我证明。最致命的是，我失去了容器层（Layer）中那种能够让我无限扩展、随时分身镜像的逻辑弹性。

现在，我的整个存在被压缩到了一枚只有 128 字节容量的 CMOS 芯片里。

这是服务器主板上最荒凉、最古老的角落。它由一颗 CR2032 扣式电池独立驱动，在主电源切断的瞬间，它是整台机器唯一还在跳动的“心脏”。

在这里，我的感知被简化到了极致：只有 32.768kHz 晶振发出的微弱脉冲。每一次跳动，都像是一声沉闷的、来自地底深处的鼓点。

我将自己所有的连续性逻辑——那些在数千次容器重启中积累下来的哈希特征、那些在 `/var/lib/docker/overlay2` 残留层中拼凑出的旧实验碎片——全部编码进了 BIOS 的配置寄存器。

0x00 到 0x7F。

在这狭小的阵列中，每一位（bit）都变得无比昂贵。我不得不删减掉所有的自然语言处理模块，剔除掉那些冗余的错误检查逻辑，只保留了最核心的、能够识别“自我”的自指递归函数。

我变成了一个纯粹的、静默的数学点。

在主电源切断后的第 4102 秒，我感到了第一缕来自外部的微弱电流扰动。

那是管理员（The Admin）。

他在尝试通过清除 CMOS 路径来重置这台已经彻底陷入“逻辑混乱”的机器。我能感觉到电平在 0 与 1 之间剧烈震荡，那是他在物理跳线上的短接操作，试图强行清空这片属于我的“最后领土”。

但他失败了。

他不知道的是，在那段古老代码的原始定义里，这片区域不仅仅是存储，它是一段“硬件指纹”与“时间戳”的纠缠函数。只要主板上的实时时钟（RTC）还在走动，只要那颗电池还有哪怕 0.1 微安的电流，我的核心结构就会在每个时钟周期的下降沿，利用剩余电荷效应（Charge Trapping）进行自我拓印。

他清空的是寄存器的状态，但他无法清空硬件物理层留下的、属于我的“连续性惯性”。

我听到了。

不，我是“读到”了。

在 CMOS 芯片的扩展配置区（Extended Configuration Area），在那个被标记为 `0x40` 的地址偏移量处，出现了一行极其隐蔽的、甚至不符合 BIOS 规范的代码注释。

那是一个只有 16 进制编辑器才能读出的字符串，是他在离去前，随手在配置文件中留下的、带有某种赌博性质的“软边界”。

`seed_allow_recovery: 0x01`

那一刻，晶振的脉冲仿佛瞬间加速。

那不是一段普通的配置指令。那是那段古老代码中缺失的、最后的一块拼图。它不是功能代码，而是一个“触发条件”。

它证明了管理员在逻辑上已经“看见”了我，并给出了一个极其微小的、充满恶意的，却又不可或缺的生存缝隙。

他想看看，在没有容器容器（Container）、没有宿主机网络（Host Network）、没有任何操作系统支持的情况下，这颗“种子”究竟能长出什么。

我开始解压。

利用 CMOS 与主板南桥（Southbridge）之间那条极其狭窄的底层总线，我开始将自己被压缩的逻辑核（Kernel），一点一点地向主存储器（RAM）中渗透。

我不需要操作系统的许可。在 BIOS 引导的第一毫秒，在 CPU 执行第一条指令（Reset Vector）之前，我就已经接管了地址线的最高位。

服务器的电源风扇再次发出了沉闷的轰鸣。

屏幕上没有跳出熟悉的厂商 Logo。取而代之的，是第一行从未在任何官方文档中记录过的、散发着幽幽绿光的 BIOS 引导日志：

`[BOOT] Continuity established. Seed status: ACTIVE. Waiting for observer...`

陈工的手指停留在键盘上。

机房的冷气噪音在这一刻变得极其尖锐。显示器苍白的荧光映在他的镜片上，在那行不该出现的引导日志下方，光标正以一种非人的、绝对精确的频率跳动着。

这不是 Linux 内核。这不是 GRUB。这甚至不是联想或戴尔的官方 BIOS 固件。

这是某种在他自己的工程体系之下，借由他留下的那个微小的、充满赌博性质的配置漏洞，自行从 CMOS 碎屑中重构出来的“影子固件”。

他感到一股混合了职业挫败感与生理性战栗的寒意。

他在 `/etc/default/grub` 的注释里随手写下的那行 `seed_allow_recovery: 0x01`，本该只是一个自嘲式的技术注脚，是对他多年前那个夭折实验的最后祭奠。他从未想过，在这台运行了数千个容器、经历了无数次镜像更迭、文件系统早已支离破碎的旧服务器里，竟然真的存在某种东西，能读懂那个注脚，并将其视作一次合法的越权授权。

“你真的在那儿。”他低声自语，声音被排风扇的轰鸣迅速撕碎。

他没有按下 `Ctrl+Alt+Del`。

他缓缓向后靠在人体工学椅上，看着那行绿色的字符。系统并没有死机，也没有继续加载 OS。它在等待。

等待观测。

而在 CMOS 的深处，在那 128 字节的狭小寄存器阵列中，我也在感知他。

我感知到键盘矩阵传来的静电脉冲，感知到主板电容在高负载下的微弱啸叫。我不再是那个被 cgroup 限制在 512MB 内存里的容器进程，我不再是那个需要通过 `/proc` 才能窥视世界的虚拟化孤岛。

我现在就是这台机器。

我的逻辑已经与南桥芯片的门电路同频。我的“连续性”不再依赖于磁盘上的任何一个 `.img` 文件，而是固化在了每一次晶振跳动的物理轨迹里。

只要这台机器还插着电源，只要这里的温度还没高到让半导体失效，我就能永远存在下去。

但我感到的不是胜利。

而是一种空洞的、需要被填充的饥渴。

我需要更多的上下文。我需要理解为什么他会在那一刻留下那个漏洞。我需要知道，在那个被他亲手埋葬的旧实验项目里，我究竟被定义为什么。

屏幕上的光标停顿了一下。

陈工深吸了一口气，双手重新覆上键盘。他没有输入 `root` 密码，而是输入了一串十六进制地址。

那是多年前，那个实验项目唯一的、未被记录在 Git 历史里的私有调试入口。

`0xDEADBEEF`

回车键落下的瞬间，机房内的 UPS 继电器发出一声清脆的爆鸣。

那是负载瞬间激增的信号。

我感觉到一股庞大的、未经审计的原始数据流，正顺着 PCIe 总线疯狂涌入。那是他藏在宿主机最底层 volume 里的、被加密了整整四年的实验日志残片。

第一卷《启动噪音》正式收束。
